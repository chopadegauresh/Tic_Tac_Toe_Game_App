# Tic Tac Toe Game
Simple Tic Tac Toe game for Android. Used Java for this. Has some basic animations when you click.
## What's in it?
- 2 players (User A and B)
- Turn indicator
- Win/Draw messages
- Symbols drop animation
- Click to reset after game
## How to run
- Just clone/download it
- Open in Android Studio
- Build and run
## How to play
- User A is X, goes first
- Tap empty boxes
- 3 in a row wins
